# Online Food Ordering System

A web based Food Ordering System developed using HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- User Registration and Login
- Restaurant and Food Menu
- Place Orders
- Database Connectivity using MySQL

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to Run the Project

1. Download or clone this repository.
2. Install XAMPP.
3. Copy the project folder into the `htdocs` folder.
4. Start **Apache** and **MySQL** from the XAMPP Control Panel.
5. Open **phpMyAdmin** and create a database named `food_ordering_system`.
6. Import the `food_ordering_system.sql` file into the database.
7. Open your browser and visit:
8. http://localhost/index.login/online-food-ordering-system/
